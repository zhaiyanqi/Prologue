## 说明
支持自定义组件：边框、势力、勾玉、技能背景等等，手动从程序中选择相册中的图片即可。<br>
可手动调节势力、勾玉、称号、武将名、技能的位置与大小。<br>
支持导出为PNG、JPG、WEBP等格式<br>
支持使用自定义字体：字体存放到sd卡根目录 ***sgs/font*** 并授予程序文件读取权限即可被检测到。<br>
